# SKILL: project-brief

## Назначение
Создание брифа для клиентского или личного проекта.

---

## Входные данные
- `client` — имя клиента / название бренда
- `project_type` — логотип / сайт / брендинг / приложение / и т.д.
- `deadline` — дедлайн
- `budget` — бюджет (опционально)
- `notes` — дополнительный контекст

---
