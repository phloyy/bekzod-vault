# SKILL: course-design

## Назначение
Создание всех материалов для курсов по графическому дизайну (и любых других).
Один skill покрывает: план курса, уроки, задания, тесты, критерии оценки, домашки, контрольные работы, дипломные проекты.

---

## Входные данные — запроси если не указано

- `course_name` — название курса (например: "Графический дизайн WePro")
- `course_slug` — короткий ID (например: `gd-wepro`, `smm-pro`)
- `block` — номер блока (0–7)
- `block_name` — название блока
- `lesson_number` — номер урока (01–47+)
- `topic` — тема урока
- `tool` — инструмент (Illustrator / CorelDRAW / Photoshop / Теория / ИИ)
- `duration` — длительность в минутах (default: 90)
- `type` — тип файла (lesson / homework / test / rubric / block-plan / course-map / diploma)

|---|---|---|---|

| 0–39 | F | Незачёт | Не выполнил |

---

## Критерии оценки работ

### Техническое выполнение (30%)
- **A:** Нет технических ошибок, файлы правильно подготовлены
- **B:** Минимальные ошибки, не влияющие на результат
- **C:** Есть ошибки, частично влияющие на результат
- **D/F:** Критические технические ошибки

### Концепция и идея (25%)
- **A:** Сильная концепция, нестандартное решение
- **B:** Чёткая концепция, логичное решение
- **C:** Концепция есть, но слабо раскрыта
- **D/F:** Концепция отсутствует

### Композиция и эстетика (25%)
- **A:** Профессиональный уровень, грамотная иерархия
- **B:** Хорошая композиция, есть мелкие недочёты
- **C:** Базовая композиция, нарушена иерархия
- **D/F:** Хаотичная композиция

### Соответствие заданию (20%)
- **A:** Полностью соответствует + дополнительные решения
- **B:** Полностью соответствует
- **C:** Частично соответствует
- **D/F:** Не соответствует заданию

## Задание

Разработать полноценный брендинг для реального или вымышленного бизнеса.

## Обязательные deliverables

- [ ] Логотип (все варианты: основной, монохром, инверсия, фавикон)
- [ ] Брендбук / Гайдлайн (min 12 страниц)
- [ ] Фирменный стиль (min 3 носителя)
- [ ] Мокапы (min 5 штук)
- [ ] Behance-кейс (min 8 слайдов)
- [ ] Презентация для защиты (10 мин)

| Этап | Что сдаётся | Дата |
|---|---|---|
| 1. Бриф и концепция | Мудборд + название + позиционирование | |
| 2. Логотип (3 варианта) | Скетчи + векторные версии | |
| 3. Фирменный стиль | Все элементы системы | |
| 4. Финальная подача | Полный пакет + Behance | |
| 5. Защита | Презентация + Q&A | |
