## Ключевые темы и навыки

### Урок 04 — Интерфейс и базовые формы
- Panels, Artboards, Workspace layout
- Rectangle, Ellipse, Polygon, Star Tool
- Трансформации: Scale, Rotate, Reflect, Shear
- Заливка и обводка: Fill / Stroke, цветовые образцы
- Горячие клавиши навигации: Z, H, Ctrl+0, Ctrl+1

### Урок 05 — Shape Builder и Pathfinder
- Shape Builder Tool: объединение, вычитание, пересечение мышью
- Pathfinder: Unite, Minus Front, Intersect, Exclude, Divide
- Compound Path: Object → Compound Path → Make
- Практика: построение логотипных форм из геометрии

### Урок 06 — Pen Tool и кривые Безье
- Anchor Points: Corner vs Smooth
- Direction Handles: управление кривизной
- Техники: click (угол), click+drag (кривая), Alt (излом)
- Convert Anchor Point Tool
- Практика: обводка объекта по растровому фото
